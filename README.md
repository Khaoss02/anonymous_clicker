# Anonymous Clicker

**Anonymous Clicker** es un juego web tipo *clicker*, simple y directo, creado como proyecto personal para practicar lógica en JavaScript, interacción con el DOM y diseño de interfaces ligeras.

**Demo en vivo:** https://anonymousclicker.netlify.app/  

---

## Descripción

El objetivo del juego es sencillo:

- Hacer clic.
- Ganar monedas.
- Mejorar progresivamente.
- Ver cómo los números suben y tu dopamina también.

Detrás de esa simplicidad hay:
- Gestión de estado del juego
- Eventos
- Actualización dinámica de la interfaz
- Base preparada para futuras mejoras

---

## Tecnologías utilizadas

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla)**
- **Netlify** para el despliegue
